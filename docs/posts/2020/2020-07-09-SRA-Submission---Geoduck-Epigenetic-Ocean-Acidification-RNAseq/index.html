2020
SRA Submission
Author

Sam White

Published

July 9, 2020

Can’t remember where it was discussed (probably lab meeting), but I created a GitHub Issue to add all of geoduck RNAseq data to NCBI Short Read Archive (SRA). Anyway, got all the remaining RNAseq data uploaded to the NCBI SRA and organized into the correct BioSamples and BioProjects.

The table below has links to each corresponding NCBI BioSample and BioProject. NCBI indicates that references should point to a particular BioProject; not individual Short Read References (SRR).

I’ve added the corresponding links to the NCBI BioProjects to our sequencing database, Nightingales (Google Sheet).
